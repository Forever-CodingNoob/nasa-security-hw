# NASA 2025 HW11: Security (Part I)

This is an archive of homework 11 of the course Network Administration and System Administration (Lab), aka NASA, in spring 2025 at NTU CSIE, with the goal of equiping students with basic computer security knowledges. The repo includes both the problem description ([NASA2025_HW11.pdf](NASA2025_HW11.pdf)) and the solutions, along with some instructions for TAs to set up crucial problem files.

I'm too lazy to write more instructions rn so please refer to [NASA2025_HW11.pdf](NASA2025_HW11.pdf) and the subdirectories for each problem for more details.
